# Aggregations and GROUP BY

## Aggregate functions

These functions compute a single result from a set of rows:

| Function | Description |
|---|---|
| `COUNT(*)` | Number of rows |
| `COUNT(col)` | Number of non-NULL values |
| `SUM(col)` | Total |
| `AVG(col)` | Average |
| `MIN(col)` | Minimum value |
| `MAX(col)` | Maximum value |

```sql
SELECT COUNT(*), AVG(price), MAX(price)
FROM products;
```

## GROUP BY

`GROUP BY` groups rows that share the same value in one or more columns, then applies aggregate functions to each group.

```sql
SELECT category_id, COUNT(*) AS nb_products, AVG(price) AS avg_price
FROM products
GROUP BY category_id;
```

> **Rule:** Every column in `SELECT` that is not inside an aggregate function **must** appear in `GROUP BY`.

## Grouping by multiple columns

```sql
SELECT country, city, COUNT(*) AS nb_customers
FROM customers
GROUP BY country, city
ORDER BY country, city;
```
